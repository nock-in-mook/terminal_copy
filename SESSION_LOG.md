1. **Command + Space** → Spotlightを開く
2. 「アクティビティモニタ」と入力してEnter
3. 検索欄に「Python」と入力
4. 該当プロセスを選択して **Command + Q** か、ツールバーの×ボタンを **Tab + Enter** で押す

もしくはもっと早い方法：

1. **Command + Space** → 「ターミナル」と入力してEnter
2. ターミナルが開いたら以下を入力：
```
pkill -f folder_launcher
```
